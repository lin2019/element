## Méthodes Popup

Fournit des méthodes pour ouvrir, fermer et naviguer entre les onglets.

### Importer

```javascript
import { popup } from 'element-ui/src/utils'

// Ou via l'espace de noms yq
yq.popup.openTab(option)
```
