## Tool Functions

Provides commonly used utility functions, including date processing, data formatting, validation, etc.

### Basic Usage

```javascript
import { tools } from 'yq-ui-code/lib/utils'

// Date formatting
tools.formatDate(new Date(), 'YYYY-MM-DD')

// Deep clone
tools.deepClone(obj)
```
